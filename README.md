# Frontend Mentor - QR Code Component

A clean, responsive, implementation of the [Frontend Mentor QR code component challenge](https://www.frontendmentor.io/challenges/qr-code-component-iux_sIO_H).

## Features

- **Semantic HTML5:** Built with semantic markup (`<main>`, `<article>`, `<footer>`).
- **Responsive Design:** Mobile-first layout that looks great on all screens.
- **Modern CSS:** Clean variables, reset guidelines, and flexible layout techniques (Flexbox) for centering.
- **Aesthetic Fonts:** Custom integration of Google Fonts (Outfit).

## Built With

- Semantic HTML5 Markup
- CSS Custom Properties (Variables)
- Flexbox Layout
- Google Fonts (Outfit)

## Getting Started

To run or view this project locally, simply:

1. Clone this repository:
   ```bash
   git clone https://github.com/nara-md/fm-qr-code-component.git
   ```
2. Open [index.html](index.html) in your browser of choice.
